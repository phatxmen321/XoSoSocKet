======Ứng dụng Xổ Số Kiến Thiết 1 server với nhiều client trên Console C/C++======
Đây là Ứng dụng giúp người dùng ứng dụng có thể tra vé số của họ có trúng hay không, từ đó server
sẽ gửi kết quả về cho các client.
//////////////////////////////////////////////////
Các client sẽ được sử dụng tuần tự.
//////////////////////////////////////////////////
Ứng dụng này dành cho bạn.
Hãy cài đặt visual studio.
//////////////////////////////////////////////////
Hướng dẫn sử dụng:
Bước 1: Mở file DoAn2_Server.sln trong file sourch lên bằng visual studio và Sau đó chạy 
chương trình.
Bước 2: Mở file DoAn2_Client.sln trong file sourch lên bằng visual studio và Sau đó chạy 
chương trình.
Bước 3: Nhập số lượng client trong màn hình Console bên phía server.
Bước 4: Mở Từng client và thực hiện các bước đăng ký, đăng nhập, tra vé số.
